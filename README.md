# (C# 코딩) 그림판 앱
## 개요
-C# 프로그래밍학습
-1줄소개: 
-사용한플랫폼: 
  -C#, .NET Windows Forms, Visual Studio, GitHub
-사용한컨트롤:
  PictureBox, ComboBox, TrackBar,Button,Label, GroupBox
-사용한기술과구현한기능:
  -Visual Studio를이용하여UI 디자인


  ## 실행화면(과제1)
  -코드의실행스크린샷과구현내용설명
  ![실행화면](img/screen1.png)
  -구현한내용(위그림참조)
    -UI 구성: PictureBox, ComboBox, TrackBar,Button,Label, GroupBox
    -픽처박스: 그림판의 캔버스 역할
    -콤보박스: 선의 색상 선택(빨강, 초록, 파랑, 검정)
    -트랙바: 선의 굵기 조절( 1~10 까지 선택 가능 )
    -버튼: 선의 스타일 선택( 직선, 사각형, 원형 )
    -그룹박스: 선 스타일 옵션 그룹화
